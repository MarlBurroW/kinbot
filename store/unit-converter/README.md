# 📐 Unit Converter Plugin

Convert between units of measurement across 10 categories. Runs entirely offline with no API calls.

## Tools

### `convert_units`
Convert a value from one unit to another. Supports abbreviations and common names.

**Examples:**
- `5 km` → `miles`
- `100 °F` → `°C`
- `2.5 GB` → `MB`
- `1 atm` → `psi`

### `list_units`
Browse available categories and units. Useful for discovering what conversions are supported.

## Configuration

No configuration needed. Works out of the box.

None required (no network access needed).
